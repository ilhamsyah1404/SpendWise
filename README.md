# 💰 SpendWise — Bijak Berbelanja, Cepat Menabung

> Aplikasi pencatat keuangan pribadi berbasis web dengan AI Advisor (Claude AI), dirancang khusus untuk pengguna Indonesia.

![SpendWise Banner](assets/banner.png)

---

## ✨ Fitur Utama

| Fitur | Deskripsi | Status |
|-------|-----------|--------|
| 📊 Dashboard | Statistik real-time, grafik mingguan, pie chart | ✅ |
| 💸 Catat Transaksi | Input cepat pengeluaran & pemasukan | ✅ |
| 🎯 Budget Planning | Atur batas per kategori, alert otomatis | ✅ |
| 📈 Laporan | Analisis periode, export data | ✅ |
| 🤖 AI Advisor | Claude AI untuk analisis & saran personal | ✅ |
| 📱 Responsive | Mobile, tablet, desktop | ✅ |

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Charts:** Chart.js v4
- **AI:** Pocket Base
- **Storage:** LocalStorage
- **Icons:** SVG inline
- **Fonts:** Sora + Space Mono (Google Fonts)
